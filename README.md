Программы: \
Программы для автопилота написаны на языке Python с использованием библиотеки kRPC

Цель проекта: \
Смоделировать полёт советского спутника "Спутник-1" в игре Kerbal Space Program, а также произвести физические и математические расчеты.

Задачи проекта:
1. Изучить историю полёта «Спутник-1», рассмотреть цели запуска, основные этапы миссии и её значение для развития космонавтики, а также проанализировать характеристики ракеты-носителя и параметры орбиты спутника. \
2. Спроектировать в Kerbal Space Program космический аппарат со спутником на борту, максимально приближённый к реальному аналогу, с учётом массы, компоновки, числа ступеней, характеристик двигателей и аэродинамики. \
3. Разработать физико-математическую модель полёта, учитывающую силы тяги, тяжести и сопротивления воздуха, изменение массы аппарата, а также влияние атмосферы и вращения планеты. \
4. Смоделировать полёт в Kerbal Space Program, выполнив запуск ракеты, выведение спутника на орбиту и отделение полезной нагрузки, зафиксировав основные параметры полёта. \
5. Реализовать вычисления по физико-математической модели программно (например, на Python), выполнить численное моделирование и построить графики изменения основных параметров во времени. \
6. Сравнить результаты моделирования в Kerbal Space Program и программной реализации, проанализировать различия и возможные причины расхождений. \
7. Составить итоговый отчёт с описанием теоретической части, процесса моделирования, полученных результатов и выводов. \
